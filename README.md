## Getting Started 

First: init config enviroment: 

- Change file `.env.test` to  `.env`

Second: install create mongodb replica set

```
  docker-compose up -d
```
Then: prisma init and push schema to database

```
  npx prisma init
  npx prisma db push 
```

Next: install dependencies and run project

```
  yarn install
  yarn dev
```
